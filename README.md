## 📌 About the Project

This is my personal portfolio website built to showcase my skills, projects, and experience as a Frontend Developer.

The portfolio is designed with a modern UI/UX approach, smooth animations, and responsive layouts to ensure a seamless experience across all devices.

---

## 🛠️ Tech Stack

- ⚛️ React
- ⚡ Vite
- ⌨️ TypeScript
- 🎨 Tailwind CSS
- 🎞️ Framer Motion
- 📦 GitHub Pages (Deployment)

---

## ✨ Features

- Fully responsive design (mobile + desktop)
- Smooth animations and transitions
- Modern glassmorphism UI style
- Sections for About, Skills, Projects, Contact
- Downloadable Resume (CV)
- Fast performance using Vite
- Deployed using GitHub Pages
